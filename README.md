## Run Locally

Clone the project

```bash
  git clone https://github.com/abdulawalarif/google_map.git
```


Go to the project directory

```bash
  cd google_map
```

Install dependencies

```bash
  flutter pub get
```

Connect a physical device or start a virtual device on your machine

```bash
  flutter run
```
